Rallax.js

On GitHub

Dead-simple parallax scrolling, without dependencies.

rallax ( '.test' , './test-image1')

Explore a variety of options, including:

  • Changing parallax speed
  • Adjusting the scale and offset of the image
  • Setting a zIndex
  • Specifying when to disable the effect, based on screen width

rallax ( '.test' , './test-image3', {

adjust: {

scale: 0.1,

offset: {

top: 0,

right: 0

})

})

})
